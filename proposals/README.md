# 📄 Propuestas

Documentos de propuesta para cambios grandes (funcionalidades nuevas, cambios de flujo, integraciones).

**Cuándo usar esto en vez de un PR con código:** cuando la idea necesita discutirse antes de construirse — más de un par de horas de trabajo, o afecta cómo opera el equipo.

**Cómo:**
1. Copia `TEMPLATE.md` → `AAAA-MM-DD-<tema-corto>.md` (ej.: `2026-07-24-ordenes-a-proveedores.md`)
2. Llénalo (con Claude Code si quieres — el CLAUDE.md del repo ya conoce este flujo).
3. Ábrelo como Pull Request en una rama `propuesta/<tema>`.
4. Se discute en el PR, se decide en la reunión del viernes, y lo aceptado entra al tablero 🗂️ y al plan de la semana siguiente.

Las ideas rápidas y el feedback del día a día NO van aquí: van al formulario 📬 SWRFM — Feedback e Ideas (Notion).
